# PaperIQ Pro 🧠📄

PaperIQ Pro is a complete full-stack AI-based Research Paper Analysis System. It allows users to upload research papers (PDFs), automatically extract and analyze content, and interact with the paper through an intelligent AI chatbot powered by Google Gemini and Retrieval-Augmented Generation (RAG).

## 🚀 Features

- **User Authentication**: Secure signup and login system to save your analysis history.
- **Smart PDF Processing**: Upload research papers and automatically extract text and metadata.
- **AI Chatbot (RAG)**: Ask questions about the uploaded paper and get accurate, context-aware answers powered by Google Gemini and local sentence transformers.
- **Advanced NLP Analysis**: Keyword extraction, summarization, and sentiment analysis using NLTK and TextBlob.
- **Visual Analytics**: Interactive data visualizations and dashboard using Matplotlib and Seaborn.
- **Chat History & History Tracking**: View past interactions and analyzed papers stored securely in MongoDB.
- **Sleek UI**: A highly responsive and aesthetic user interface built with Streamlit.

## 🛠️ Tech Stack

- **Frontend**: Streamlit
- **Backend**: Python
- **Database**: MongoDB (via `pymongo`)
- **AI & ML**: Google Generative AI (Gemini), `sentence-transformers`, `scikit-learn`
- **NLP**: `nltk`, `textblob`, `transformers`
- **PDF Processing**: `PyPDF2`, `pymupdf` (fitz)
- **Data Visualization**: `matplotlib`, `seaborn`, `pandas`, `numpy`

## 📋 Prerequisites

Before you begin, ensure you have the following installed:
- Python 3.8+
- MongoDB (Local instance or MongoDB Atlas)
- Google Gemini API Key
